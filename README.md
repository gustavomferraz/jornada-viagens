# 🌍 Jornada Viagens

Projeto desenvolvido como prática de desenvolvimento front-end, simulando um site de pacotes de viagens.

## 📌 Sobre o projeto

O **Jornada Viagens** é um site fictício que simula a apresentação e organização de pacotes de viagens.
O objetivo do projeto é praticar conceitos fundamentais de desenvolvimento web, como estruturação de páginas, estilização e organização de código.

---

## 🚀 Funcionalidades

* Exibição de pacotes de viagens
* Layout organizado e responsivo
* Estrutura de páginas com HTML e CSS
* Uso de fontes personalizadas
* Organização de arquivos e pastas

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

---

## 📂 Estrutura do projeto

```bash
📁 jornada-viagens
 ┣ 📁 CSS
 ┣ 📁 fontes
 ┣ 📄 index.html
```

---

## 💻 Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/gustavomferraz/jornada-viagens.git
```

2. Acesse a pasta do projeto:

```bash
cd jornada-viagens
```

3. Abra o arquivo `index.html` no navegador
